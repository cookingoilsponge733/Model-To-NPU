# Rooted phone sample

**Languages:** [English](README_EN.md) | [Русский](README_RU.md)

This folder contains a small, real-world rooted phone-side sample captured from `/data/local/tmp/sdxl_qnn`.

It is intentionally limited to lightweight artifacts:

- helper shell scripts;
- QNN backend JSON configs;
- context-generation log files;
- a couple of generated PNG results.

Model weights, QNN context binaries, and large runtime libraries are **not** included here.
